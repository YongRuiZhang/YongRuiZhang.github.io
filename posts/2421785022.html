JetBrains（IDEA、PyCharm等）下载教程

介绍

JetBrains是一家捷克的软件开发公司，该公司位于捷克的布拉格，并在俄罗斯的圣彼得堡及美国麻州波士顿都设有办公室，该公司最为人所熟知的产品是Java编程语言开发撰写时所用的集成开发环境：IntelliJ IDEA。

产品

IDE

  • Fleet - 下一代JetBrains 的 IDE
  • AppCode - 适用于 iOS/macOS 开发的智能 IDE
  • CLion - 支持 C 和 C++ 的跨平台 IDE
  • DataGrip - 多种数据库，一个工具
  • GoLand - 专为 Go 开发者设计的跨平台 IDE
  • IntelliJ IDEA - 功能强大，符合人体工程学的 JVM IDE
  • PHPStorm - 高效智能的 PHP 集成开发工具
  • PyCharm - 智能 Python 集成开发工具
  • Rider - 快速且强大的跨平台 .NET IDE
  • RubyMine - 最智能的 Ruby 与 Rails IDE
  • WebStorm - 最智能的 JavaScript IDE

.NET

  • ReSharper - 适用于.NET开发者的Visual Studio扩展
  • dotCover - .NET 单元测试工具
  • dotMemory - .NET 内存分析器
  • dotPeek - 免费 .NET 反编译器和浏览器
  • dotTrace - .NET 性能分析器

团队工具

  • Space - 集成的团队环境
  • TeamCity - 安装即用的强大持续集成工具
  • Upsource - 代码审查、项目分析
  • YouTrack - 为敏捷团队打造的项目管理工具
  • Datalore - 提供智能编码辅助的在线数据科学笔记本
  • Hub - YouTrack、TeamCity 和 Upsource 的连接器

教育优惠下载

访问官网：中国区官网链接

首先点击右上角先注册或登录JetBrains账号,或者直接点击登录链接

关于注册：

这里用什么邮箱注册都可以，用什么方式注册也都是可以的。

注册登录成功后界面如下（我这里是已经有了这些产品的许可证，如果是未申请的人应该不是这样）：

等会儿申请成功后，就可以点击产品下面的链接直接下载了。

登录后再访问：学生或教师认证链接

向下滑动页面，到如图位置，点击立即申请：

点击后跳转到如下界面，填写信息即可（这里的邮箱一定要是学校开户的邮箱（不同学校开户方式可能不同，请自行搜索如何开户）才行，不能是QQ邮箱、谷歌邮箱这种）：

申请成功登录账号，点击链接下载即可。例如点击IntelliJ IDEA Ultimate

再点击Download，跳转到如下界面：

选择对应的版本下载即可。

下载好产品后，需要登录刚才申请的账号，这样才会把许可证加载到产品中长期使用。

这种方式每年都需要再去申请一次，再许可证到期前一周官方会发邮件提示到。
